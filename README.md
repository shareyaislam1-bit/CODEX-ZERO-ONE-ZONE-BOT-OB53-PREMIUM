- ✅ Role assignment — Leader / Co-Leader / Member
- ✅ Automatic guild activity tracker

## 📦 Installation

**Step 1 — Clone the repo**
```bash
git clone https://github.com/YOUR_USERNAME/freefire-guild-bot-ob53.git
cd freefire-guild-bot-ob53
```

**Step 2 — Install dependencies**
```bash
pip install -r requirements.txt
```

**Step 3 — Configure**
```python
# Edit config.py
BOT_TOKEN = "your_bot_token"
GUILD_ID  = "your_guild_id"
```

**Step 4 — Run**
```bash
python main.py
```
